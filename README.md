## 💰 Controle Real — Controle Financeiro Pessoal

Um aplicativo web completo para gestão financeira pessoal, desenvolvido com Flask, com foco em clareza, automação e experiência moderna.
O Controle Real foi criado para ajudar usuários a entender melhor sua vida financeira, automatizando cálculos importantes e oferecendo uma visão clara de gastos, metas e saúde financeira.
A proposta é simples: transformar dados financeiros em decisões inteligentes.

🔗 Acesse agora:
👉 https://controle-financeiro-02by.onrender.com

## 🚀 Funcionalidades
## 💵 Gestão de Salário
- Cálculo automático de salário líquido
- Desconto de INSS e IRRF por faixas atualizadas
- Visualização clara do valor disponível
## 💳 Controle de Despesas
- Cadastro rápido e intuitivo
- Categorização inteligente:
Alimentação, Transporte, Moradia, Lazer, Saúde, Educação, etc.
- Histórico completo de gastos
## 🎯 Metas Financeiras
- Criação de objetivos personalizados
- Acompanhamento de progresso em tempo real
- Indicadores visuais de evolução
## 📊 Dashboard Interativo
- Gráficos de gastos por categoria
- Evolução financeira mensal
- Score de saúde financeira
## 🌙 Interface Moderna
- Tema claro/escuro
- Layout responsivo
- Design inspirado em apps financeiros reais (ex: Nubank)

### 📊 Regras de Negócio

**💰 Cálculo de Salário**
- INSS progressivo por faixa salarial
- IRRF conforme tabela oficial
- Cálculo automático do líquido

**📉 Score Financeiro**
- Baseado na proporção de gastos vs renda
- Penalizações por gastos excessivos e saldo negativo

**🎯 Metas**
- Cálculo automático de progresso (%)
- Feedback visual de conclusão

### Como Executar Localmente

**Clone o repositório**
- git clone https://github.com/SEU-USUARIO/controle-real.git

**Acesse a pasta**
- cd controle-real

**Instale as dependências**
- pip install -r requirements.txt

**Execute o projeto**
- python run.py
